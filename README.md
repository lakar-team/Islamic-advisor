**Islamic Advisor** is a modern, AI-powered guidance platform designed to provide compassionate and accurate Islamic insights. Built with a "Digital Sanctuary" aesthetic, it combines ancient wisdom with state-of-the-art AI to offer a private, respectful space for spiritual consultation.

## ✨ The Experience

- 🌿 **Sheikh Chat**: An advanced AI interface that provides empathetic, context-aware answers to complex questions, rendered with elegant, high-fidelity UI components.
- 📚 **Knowledge Architecture**: A structured library system for deep-diving into specific topics of Fiqh, Hadith, and spiritual development.
- 🔒 **Private Consultation**: Your privacy is paramount. All conversation history and personal settings are stored exclusively on your local device, ensuring total data sovereignty.
- 📊 **Community Pulse**: Transparent, live impact metrics and rolling reviews that reflect the platform's commitment to community support.

## 🚀 Technical Excellence

- **Core Engine**: React 18 / Vite / TypeScript
- **Intelligence Layer**: Google Gemini 2.0 Flash via secure Edge Functions.
- **Infrastructure**: Cloudflare Pages & Functions (Edge Runtime) for ultra-low latency worldwide.
- **Security**: Hardened client-side rate limiting and protocol-level abuse prevention.

## 🛠️ Getting Started

### 1. Installation
```bash
npm install
```

### 2. Ignition
Start the development environment:
```bash
npm run dev
```

### 3. Deployment
The platform is optimized for seamless deployment to Cloudflare Pages:
```bash
npm run build
```

## 📦 Project Structure

- `/src/components`: Premium UI components (SheikhChat, Library, ReviewSystem).
- `/src/theme`: The "Digital Sanctuary" design system (Emerald & Slate).
- `/functions/api`: Secure edge-side AI integration logic.
